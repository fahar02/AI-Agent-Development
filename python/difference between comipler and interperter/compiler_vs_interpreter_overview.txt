                        Welcome to Programming Basics!

This learning series will help you understand two fundamental concepts in 
programming: Compilers and Interpreters.

--------------------------------------------------------------------------------

--------------------------------------------------------------------------------

This guide is divided into separate files, each covering a specific topic:

1. what_is_compiler.txt
   - Understanding what a compiler is
   - Real-life analogies to make it simple
   - Basic concepts explained for beginners

2. what_is_interpreter.txt

   - How it differs from a compiler
   - Simple explanations with examples

3. key_differences.txt

   - Clear differences explained
   - Comparison table for quick reference

4. how_they_work.txt
   - Step-by-step internal working
   - Flow diagrams (ASCII format)
   - Detailed execution process

5. real_world_examples.txt
   - Languages that use compilers (C, C++, Java)
   - Languages that use interpreters (Python, JavaScript)
   - Practical code examples

6. pros_and_cons.txt
   - Advantages of compilers
   - Advantages of interpreters
   - Disadvantages of each
   - When each approach is better

7. when_to_use.txt
   - Practical guidance on choosing
   - Use cases for compilers
   - Use cases for interpreters
   - Decision-making tips

8. summary_and_revision.txt
   - Quick summary of all concepts
   - Revision checklist
   - Key points to remember
