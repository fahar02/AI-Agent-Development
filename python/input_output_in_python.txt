1. WHAT IS INPUT AND OUTPUT?
================================================================================

DEFINITION:

Input:
    Input is the data or information that we give to a program. It's like 
    asking a question to someone and waiting for their answer. When a program 
    needs information from the user, it takes INPUT.

Output:
    Output is the result or information that a program shows to the user. 
    It's like when someone answers your question. When a program wants to 
    show something to the user, it displays OUTPUT.


REAL-LIFE ANALOGY:

    Think of a calculator:
    
    INPUT  → You press buttons (numbers and operators)
    PROCESS → Calculator performs calculation
    OUTPUT → Calculator shows the result on screen

    Similarly in Python:
    
    INPUT  → User types data (name, age, numbers, etc.)
    PROCESS → Python program processes that data
    OUTPUT → Program displays results on screen


WHY INPUT/OUTPUT IS IMPORTANT:

    Without input and output, programs would be useless!
    
    - INPUT allows programs to be interactive and work with different data
    - OUTPUT allows programs to communicate results to users
    - Together, they make programs useful and user-friendly

    Example: A login system needs your username (INPUT) and shows whether 
    login was successful (OUTPUT).

2. TAKING INPUT IN PYTHON
================================================================================

THE input() FUNCTION:

    Python provides a built-in function called input() to take input from users.

    Basic Syntax:
        variable_name = input()
    
    With a prompt message:
        variable_name = input("Enter something: ")


IMPORTANT: DEFAULT STRING BEHAVIOR

    The input() function ALWAYS returns data as a STRING (text), even if 
    you enter numbers!

    Example:
        age = input("Enter your age: ")
        
        If you type: 25
        Python stores it as: "25" (a string, not a number)


EXAMPLE 1: Taking Name Input

    name = input("What is your name? ")
    print("Hello, " + name)

    When you run this:
    
    Output on screen:
        What is your name? 
    
    User types:
        John
    
    Output on screen:
        Hello, John


EXAMPLE 2: Taking Multiple Inputs

    name = input("Enter your name: ")
    city = input("Enter your city: ")
    country = input("Enter your country: ")
    
    print("You are " + name + " from " + city + ", " + country)

    Sample Run:
    
        Enter your name: Sarah
        Enter your city: Mumbai
        Enter your country: India
        You are Sarah from Mumbai, India


TYPE CONVERSION:

    Since input() always returns a string, we need to convert it to numbers 
    when doing mathematical operations.

    Common Conversions:
    
        int()   → Converts to integer (whole number)
        float() → Converts to decimal number
        str()   → Converts to string (text)


EXAMPLE 3: Taking Number Input and Converting

    Wrong Way (causes error in calculations):
    
        num1 = input("Enter first number: ")
        num2 = input("Enter second number: ")
        result = num1 + num2
        print(result)
        
        If you enter 5 and 3:
        Output: 53  (strings concatenated, not added!)
    
    
    Correct Way:
    
        num1 = input("Enter first number: ")
        num2 = input("Enter second number: ")
        num1 = int(num1)
        num2 = int(num2)
        result = num1 + num2
        print(result)
        
        If you enter 5 and 3:
        Output: 8  (numbers added correctly!)
    
    
    Better Way (convert immediately):
    
        num1 = int(input("Enter first number: "))
        num2 = int(input("Enter second number: "))
        result = num1 + num2
        print(result)


EXAMPLE 4: Using float() for Decimal Numbers

    height = float(input("Enter your height in meters: "))
    weight = float(input("Enter your weight in kg: "))
    
    bmi = weight / (height * height)
    print("Your BMI is:", bmi)

    Sample Run:
    
        Enter your height in meters: 1.75
        Enter your weight in kg: 70
        Your BMI is: 22.857142857142858

3. DISPLAYING OUTPUT IN PYTHON
================================================================================

THE print() FUNCTION:

    Python uses the print() function to display output on the screen.

    Basic Syntax:
        print(value)
        print(value1, value2, value3)


EXAMPLE 1: Simple Output

    print("Hello, World!")
    
    Output:
        Hello, World!


EXAMPLE 2: Printing Variables

    name = "Alice"
    age = 25
    
    print(name)
    print(age)
    
    Output:
        Alice
        25


EXAMPLE 3: Printing Multiple Values

    name = "Bob"
    age = 30
    city = "Delhi"
    
    print(name, age, city)
    
    Output:
        Bob 30 Delhi
    
    Notice: Python automatically adds spaces between values!


USING THE sep PARAMETER:

    The sep parameter controls what appears between multiple values.
    Default is a space " ".

    Example:
    
        print("Apple", "Banana", "Cherry")
        Output: Apple Banana Cherry
        
        print("Apple", "Banana", "Cherry", sep=", ")
        Output: Apple, Banana, Cherry
        
        print("Apple", "Banana", "Cherry", sep=" | ")
        Output: Apple | Banana | Cherry
        
        print("2024", "04", "01", sep="-")
        Output: 2024-04-01
        
        print("Hello", "World", sep="")
        Output: HelloWorld


USING THE end PARAMETER:

    The end parameter controls what appears at the end of print().
    Default is a newline "\n" (moves to next line).

    Example:
    
        print("Hello")
        print("World")
        
        Output:
            Hello
            World
        
        
        print("Hello", end=" ")
        print("World")
        
        Output:
            Hello World
        
        
        print("Loading", end="...")
        print("Done!")
        
        Output:
            Loading...Done!


EXAMPLE 4: Combining sep and end

    print("Name:", "John", sep=" ", end=" | ")
    print("Age:", 25, sep=" ", end=" | ")
    print("City:", "London")
    
    Output:
        Name: John | Age: 25 | City: London

4. FORMATTING OUTPUT
================================================================================

WHY FORMAT OUTPUT?

    Formatting makes output look professional and readable. Instead of:
    
        The price is 100 rupees
    
    We can format it as:
    
        The total price is Rs. 100.00


METHOD 1: f-STRINGS (RECOMMENDED - MODERN AND EASY)

    f-strings (formatted string literals) are the easiest and most readable 
    way to format output in Python 3.6+.

    Syntax:
        f"text {variable} more text"
    
    Notice the 'f' before the quote and curly braces {} around variables.


EXAMPLE 1: Basic f-string

    name = "Emma"
    age = 22
    
    print(f"My name is {name} and I am {age} years old.")
    
    Output:
        My name is Emma and I am 22 years old.


EXAMPLE 2: f-string with Expressions

    length = 10
    width = 5
    
    print(f"The area of rectangle is {length * width} square units.")
    
    Output:
        The area of rectangle is 50 square units.


EXAMPLE 3: f-string with Formatting

    price = 1234.5
    
    print(f"Price: Rs. {price:.2f}")
    
    Output:
        Price: Rs. 1234.50
    
    Explanation: .2f means 2 decimal places for float


EXAMPLE 4: Multiple Variables in f-string

    product = "Laptop"
    quantity = 3
    price = 45000
    total = quantity * price
    
    print(f"Product: {product}")
    print(f"Quantity: {quantity}")
    print(f"Price per unit: Rs. {price}")
    print(f"Total Amount: Rs. {total}")
    
    Output:
        Product: Laptop
        Quantity: 3
        Price per unit: Rs. 45000
        Total Amount: Rs. 135000


METHOD 2: format() METHOD

    The format() method is another way to format strings.

    Syntax:
        "text {} more text".format(value)


EXAMPLE 1: Basic format()

    name = "David"
    age = 28
    
    print("My name is {} and I am {} years old.".format(name, age))
    
    Output:
        My name is David and I am 28 years old.


EXAMPLE 2: format() with Index Numbers

    print("I like {0} and {1}".format("pizza", "burger"))
    print("I like {1} and {0}".format("pizza", "burger"))
    
    Output:
        I like pizza and burger
        I like burger and pizza


EXAMPLE 3: format() with Named Arguments

    print("Name: {n}, Age: {a}".format(n="Tom", a=35))
    
    Output:
        Name: Tom, Age: 35


METHOD 3: % FORMATTING (OLD STYLE - BASIC INTRO)

    This is the oldest method, similar to C language printf.

    Common Format Specifiers:
        %s → String
        %d → Integer
        %f → Float

    Example:
    
        name = "Lisa"
        age = 27
        
        print("Name: %s, Age: %d" % (name, age))
        
        Output:
            Name: Lisa, Age: 27

6. COMMON MISTAKES AND HOW TO AVOID THEM
================================================================================

MISTAKE 1: Forgetting Type Conversion

    Wrong:
        age = input("Enter age: ")
        next_year_age = age + 1
        
        Error: TypeError: can only concatenate str (not "int") to str
    
    Correct:
        age = int(input("Enter age: "))
        next_year_age = age + 1


MISTAKE 2: Converting Non-numeric Input to int/float

    Wrong:
        name = int(input("Enter your name: "))
        
        If user enters "John":
        Error: ValueError: invalid literal for int() with base 10: 'John'
    
    Correct:
        name = input("Enter your name: ")


MISTAKE 3: Not Handling Division by Zero

    Wrong:
        num = int(input("Enter a number: "))
        result = 100 / num
        
        If user enters 0:
        Error: ZeroDivisionError: division by zero
    
    Correct:
        num = int(input("Enter a number: "))
        if num != 0:
            result = 100 / num
            print(result)
        else:
            print("Cannot divide by zero!")


MISTAKE 4: Mixing String Concatenation and Numbers

    Wrong:
        age = 25
        print("Age: " + age)
        
        Error: TypeError: can only concatenate str (not "int") to str
    
    Correct Options:
        age = 25
        print("Age: " + str(age))
        print("Age:", age)
        print(f"Age: {age}")


MISTAKE 5: Forgetting Quotes in print()

    Wrong:
        print(Hello)
        
        Error: NameError: name 'Hello' is not defined
    
    Correct:
        print("Hello")


MISTAKE 6: Using Comma Instead of Plus for String Concatenation

    Different Results:
        name = "John"
        
        print("Hello " + name)
        Output: Hello John
        
        print("Hello ", name)
        Output: Hello  John
        
        Notice the extra space with comma!

7. ASCII DIAGRAM: INPUT-OUTPUT FLOW
================================================================================

    +------------------+
    |                  |
    |   USER TYPES     |
    |   INPUT DATA     |
    |                  |
    +--------+---------+
             |
             | (Data flows into program)
             v
    +------------------+
    |                  |
    |  PYTHON PROGRAM  |
    |   PROCESSES      |
    |   THE DATA       |
    |                  |
    +--------+---------+
             |
             | (Result flows out)
             v
    +------------------+
    |                  |
    |  OUTPUT SHOWN    |
    |  ON SCREEN       |
    |                  |
    +------------------+


    Example Flow:
    
    USER INPUT: "25"
         |
         v
    input() function receives "25" as string
         |
         v
    int() converts "25" to 25 (number)
         |
         v
    Program calculates: 25 + 5 = 30
         |
         v
    print() displays: "30"
         |
         v
    USER SEES: 30 on screen

8. TIPS FOR WRITING CLEAN INPUT/OUTPUT CODE
================================================================================

TIP 1: Always Use Descriptive Prompts

    Bad:
        x = input()
    
    Good:
        age = input("Please enter your age: ")


TIP 2: Convert Input Immediately When Needed

    Good Practice:
        age = int(input("Enter age: "))
        
    Instead of:
        age = input("Enter age: ")
        age = int(age)


TIP 3: Use f-strings for Modern Python

    f-strings are clearer and faster:
        print(f"Hello {name}, you are {age} years old")


TIP 4: Add Blank Lines for Readability

    print("Welcome!")
    print()
    name = input("Enter name: ")
    print()
    print(f"Hello, {name}!")


TIP 5: Validate User Input When Possible

    age = input("Enter age: ")
    
    if age.isdigit():
        age = int(age)
        print(f"Your age is {age}")
    else:
        print("Please enter a valid number!")


TIP 6: Use Meaningful Variable Names

    Bad:
        x = input("Enter name: ")
        y = input("Enter age: ")
    
    Good:
        name = input("Enter name: ")
        age = input("Enter age: ")

KEY POINTS TO REMEMBER:

   - Use end parameter to change line ending

3. FORMATTING:

   - % formatting: "text %s" % variable (OLD)

4. COMMON MISTAKES:
   - Forgetting type conversion
   - Not handling invalid input
   - Mixing string and number operations

5. BEST PRACTICES:
   - Use descriptive prompts
   - Convert input immediately
   - Use f-strings for formatting
   - Validate user input
   - Use meaningful variable names
