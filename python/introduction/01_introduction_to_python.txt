What is Python?
---------------

Python is a programming language that helps us talk to computers. Just like 
we use English or Hindi to talk to people, we use Python to give instructions 
to computers.

Think of it this way:
- When you want your friend to make tea, you say: "Boil water, add tea leaves,
  add sugar, add milk"
- When you want a computer to do something, you write instructions in Python

Why Python?
-----------

Python is special because:

- Easy to Read: Python code looks almost like English sentences
- Beginner Friendly: You don't need to be a genius to start
- Powerful: You can build websites, games, AI programs, and much more
- Popular: Millions of programmers use Python worldwide
- Free: You don't have to pay anything to use Python

Real-Life Uses of Python:
-------------------------

- YouTube uses Python to manage videos
- Instagram uses Python for its backend
- NASA uses Python for space calculations
- Netflix uses Python to recommend movies
- Scientists use Python to analyze data
